# Plataforma de Afiliados Dynamo

Plataforma de gestão de afiliados e parceiros, integrada à Shopify.

## Visão Geral

Sistema para gerenciamento do programa de afiliados, incluindo:

- Lógica de afiliados e parceiros
- Integração com Shopify
- Geração e rastreamento de cupons
- Regras de comissão e progressão
- Painel administrativo
- Integrações com WhatsApp e e-mail

## Stack Tecnológica

- **Runtime:** Node.js
- **Frontend:** React / Next.js
- **Linguagem:** TypeScript
- **Banco de dados:** Supabase (Prisma como ORM)
- **E-commerce:** Shopify

## Status

Projeto em desenvolvimento inicial. A estrutura completa (frontend, backend, banco de dados e workflows) será adicionada nas próximas etapas.
